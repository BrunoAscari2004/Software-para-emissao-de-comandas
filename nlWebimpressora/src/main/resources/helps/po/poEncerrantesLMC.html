Manutenção dos encerrantes

Nesta tela é possível alterar as quantidades abertura e fechamento.

Para trazer valores em tela é necessário realizar o filtro das informações desejadas. Apenas o campo unidade é obrigatório.

Dois cliques sobre os campos de quantidade habilitam o valor para edição. Pressionar a tecla 'enter' para finalizar a edição do valor. E por fim clicar em 'Salvar' para atualizar os valores ajustados.
