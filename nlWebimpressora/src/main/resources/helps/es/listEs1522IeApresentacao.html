ES - Apresentações dos itens das etiquetas

Esta página permite a consulta e inclusão de valores de apresentação dos itens das etiquetas.

Pesquisa

Campo Conteúdo
Item Código do item conforme a máscara.

Listagem

Campo Conteúdo
UM Código da unidade de medida para apresentação do item nas etiquetas. Não é permitido cadastrar duas unidade de medida iguais para o mesmo item.
Descrição Descrição da unidade de medida de apresentação do item.
FC Fator de conversão da unidade de medida de apresentação informada para o item.
% Percentual de sugestão para gerar a quantidade de etiquetas. Exemplo: Se neste campo o percentual for de 50% e a quantidade de etiquetas disponível for 10, então ele levará como sugestão a impressão de 5 etiquetas que corresponte aos 50% informados. Valor pode ser atualizado através do campo.
Botão excluir Exclui o registro.
Botão novo Cria um novo registro.

Manutenção

Campo Conteúdo
Item Código do item conforme a máscara.
UM Código da unidade de medida para apresentação do item nas etiquetas. Não é permitido cadastrar duas unidade de medida iguais para o mesmo item.
Fator de conversão Fator de conversão da unidade de medida de apresentação informada para o item.
%Etiquetas Percentual de sugestão para gerar a quantidade de etiquetas. Exemplo: Se neste campo o percentual for de 50% e a quantidade de etiquetas disponível for 10, então ele levará como sugestão a impressão de 5 etiquetas que corresponte aos 50% informados. Valor pode ser atualizado através do campo.
Etiquetas Indicador de qual unidade de medida deve ser utilizada como default na geração das etiquetas para o item. É necessário possuir uma unidade de medida definida como default.
Botão voltar Volta para a lista de valores de apresentação do item.
Botão duplicar Duplica o registro.
Botão excluir Exclui o registro.
Botão salvar Salva o registro.
