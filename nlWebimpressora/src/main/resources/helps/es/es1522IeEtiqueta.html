ES_1522_IE_ETIQUETA

Impressão de etiquetas

O objetivo deste programa é permitir a emissão de etiquetas para os itens em estoque com base em uma nota de entrada ou com base na quantidade em estoque.
Este programa é composto por duas pastas. A pasta Notas é para emissão de etiquetas com base em uma nota de entrada. A pasta Estoques é para emissão das etiquetas com base no estoque do item..
As duas pastas são independentes, não existe relação entre elas, ou seja, é permitido imprimir etiquetas com os itens da nota de entrada juntamente com as etiquetas do item com base na quantidade em estoque, tudo ao mesmo tempo.

Cabeçalho

Grupo de unidades

Código do grupo de unidades para seleção das notas de entrada

Formulário

Código do formulário a ser utilizado para impressão das etiquetas

Usuário

Caso a nota ou o código de barras consultado possuam a informação do usuário que gerou, esta informação será mostrada neste campo.

Notas

Nesta guia deverá ser informado os dados da nota fiscal de entrada, onde serão listados todos os seus itens para seleção de quais devem emitir etiquetas

Dados da nota fiscal de entrada

Número da nota fiscal de entrada e série

Nota

Código do grupo de unidades para seleção das notas de entrada

Sequência

Número da sequência da nota de entrada e código da máquina de inclusão da nota

Recebimento

Data de recebimento da nota fiscal de entrada

Emitente

Código completo do emitente da nota fiscal de entrada e nome

Unidade

Código da unidade da nota fiscal de entrada e descrição

Itens da nota fiscal de entrada

Nesta parte deve ser informado quais itens da nota devem emitir etiquetas, bem com a quantidade de etiquetas a emitir, unidade de medida utilizada e quantidade de itens em cada etiqueta. Serão listados os itens agrupando por Código do item, número do lote e data de validade

Item

Código do item da nota fiscal de entrada listado conforme a máscara corrente

UM Estoque

Código do item da nota fiscal de entrada listado conforme a máscara corrente

Lote

Número do lote do item na nota fiscal de entrada

Validade

Data de validade do item na nota fiscal de entrada

Qtde. nota

Quantidade do item na nota fiscal de entrada, agrupando por item, lote e data de validade

%

Percentual de sugestão da quantidade de etiquetas que devem ser geradas. Esse campo vem preenchido com o percentual informado como default no programa de manutenção das apresentações do item.

Qtde. Etiqueta

Quantidade de etiquetas a serem emitidas para o item.É calculada de acordo com a quantidade do item na nota fiscal de entrada levando em consideração também o percentual de sugestão informado.

UM Etiq.

Código da unidade de medida a ser utilizada para emitir as etiquetas do item. Esse campo vem preenchido com a unidade de medida informada como default no programa de manutenção das apresentações do item.

Alterar

Ao clicar nesse botão será aberta uma nova tela onde poderão ser informadas as quantidade de etiquetas que se deseja emitir para cada unidade de medida cadastrada para o item no programa de manutenção das apresentações do item mais a unidade de medida do item no estoque.
Observação: Se informado as etiquetas a serem impressão nesta tela, não serão mais considerados os dados informados na tela principal para os itens da nota de entrada. O programa irá obedecer o que estiver informado na tela das unidade de medida de apresentação e se essa estiver em branco será considerado as informações da tela principal.

Gerar

Indicador de quais itens devem gerar etiquetas, se este indicador estiver desmarcado, não serão emitidas as etiquetas para este item

Item

Descrição do item da nota fiscal de entrada

Fator de conversão

Fator de conversão da unidade de medida utilizada para emissão das etiquetas do item

Estoques

Nesta parte pode ser selecionados os itens atraver de um leitor de código de barras ou informando o item manualmente, após informado item, serão listadas as quantidades de estoque do item, agrupando essas quantidades pelo lote e pela data de validade do item. Para em seguida poder informar as quantidades de etiquetas a serem geradas.

Dados do item no estoque

Nesta parte deve ser informado o código do item ou carregado pelo código de barras, e também selecionar o tipo de estoque a ser listado, atual ou disponível.

Leitura do código de barras

Neste campo pode ser lido o código do item do estoque através do seu código de barras, após lido o código de barras o código do item será carregado no campo item através da máscara do código de barras.

Item

Código do item do estoque, esse campo pode ser informado quando o mesmo não foi lido pelo código de barras
Descrição do item da nota fiscal de entrada.

Estoque

Tipo de estoque a ser listado. Pode ser: 1 - Disponível ou 2 - Atual.

Quantidade

Quantidade do item no estoque de acordo com o tipo de estoque informado.

Múltiplo

Quantidade do lote múltiplo para reposição. Esta informação está no cadastro de parâmetros do item (ce_pars_calculo). Busca o múltiplo de qualquer unidade do grupo informado. Se não encontrar buscar o múltiplo para a empresa.

Marca

Descrição da máscara do item correspondente a marca do mesmo. A marca está na máscara informada no parâmetro Cirurgica - 14.

Lote

Número do lote do item no estoque.

Validade

Data de validade do item no estoque.

Qtde. Estoque

Quantidade do item no estoque para o lote e data de validade.

UM Etiqueta

Unidade de medida do item para ser utilizada na impressão das etiquetas. É possível selecionar apenas as unidade de medida que tiverem sido cadastradas no programa de manutenção das apresentações do item.

FC

Fator de conversão do item na unidade de medida informada. Esse campo é informado no programa de manutenção das apresentações do item.

%

Percentual de sugestão da quantidade de etiquetas que devem ser geradas. Esse campo vem preenchido com o percentual informado no programa de manutenção das apresentações do item da unidade de medida informada.

Qtde. Etiqueta

Quantidade de etiquetas a serem emitidas para o item, é calculada de acordo com a quantidade do item no estoque levando em consideração também o percentual de sugestão informado.

Qtde. Item

Quantidade de itens na etiqueta, calculado pela quantidade de etiquetas vezes o fator de conversão.

Saldo

Quantidade de itens que não geraram etiqueta. Calculado pela quantidade em estoque menos a quantidade de etiquetas vezes o fator de conversão.

Alterar

Ao clicar nesse botão será aberta uma nova tela onde poderão ser informadas as quantidade de etiquetas que se deseja emitir para cada unidade de medida cadastrada para o item no programa de manutenção das apresentações do item mais a unidade de medida do item no estoque.
Observação: Se informado as etiquetas a serem impressão nesta tela, não serão mais considerados os dados informados na tela principal para os itens em estoque. O programa irá obedecer o que estiver informado na tela das unidade de medida de apresentação e se essa estiver em branco será considerado as informações da tela principal.

Gerar saldo

Indicador se deve gerar etiquetas com a quantidade saldo dos itens, nesse caso será gerada uma etiqueta para cada item saldo, a unidade de medida utilizada será a unidade de medida do estoque do item

Gerar

Indicador de quais itens devem gerar etiquetas, se este indicador estiver desmarcado, não serão emitidas as etiquetas para este item

OK

Caso algum registro seja marcado para gerar etiquetas, será aberto uma tela solicitando que seja informado o usuário que irá gerar.

Cancelar

Sai do programa sem fazer a emissão das etiquetas.

Usuário

Código do usuário que fez a impressão das etiquetas. Somente serão válidos os usuários que possuirem vínculo com uma pessoa ativa no sistema.

OK

Valida o usuário informado conforme explicação acima.
Chama o programa para emissão das etiquetas e limpa a tela para nova emissão.

Cancelar

Fecha a tela do usuário sem fazer a emissão das etiquetas.
